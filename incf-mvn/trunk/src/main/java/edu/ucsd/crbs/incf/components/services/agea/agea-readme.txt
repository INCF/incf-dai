AGEAServiceBean.java: 
Service Bean is a middle tier in the multitier architecture that connects the Controller layer with the Data Layer.

AGEAServiceController.java: 
ServiceController takes parameter from the form, reads it, processes it, and finally returns the results back to the final users.

AGEAServiceDAOImpl.java:
DAOImpl is data layer that makes connection to the datAGEAse, reads, and writes data from/to the datAGEAse.

AGEAServiceVO.java:
VO is value object that serves a role of getters/setters in the classes.
